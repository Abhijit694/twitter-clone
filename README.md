# MERN Twitter Clone

A full-stack Twitter clone built with MongoDB, Express, React, and Node.js.

## Project Structure
- **/frontend**: React.js application.
- **/backend**: Node.js and Express API.

## How to Run
1.Clone the repo
2.Navigate to backend in terminal(cd backend)
3.npm install
4.npm run dev

5.Navigate to frontend in terminal(cd frontend)
6.npm install
7.npm run dev
8.In browser open url 'http://localhost:5173'
